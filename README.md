HexMilkTea’s Café – Final Project
Savannah McDowell

This website represents my streaming brand, HexMilkTea’s Café. It is a cozy horror themed site that includes information about my stream, the games I play, my streaming schedule, and a contact form.

Pages included:

index.html – Homepage introducing the brand and overall vibe.
about.html – Explains the purpose of the café and what viewers can expect.
games.html – Highlights the types of games streamed and includes a table.
schedule.html – Displays a weekly streaming schedule using a table.
contact.html – Includes a contact form for visitors to send a message.

External Files:

css/styles.css – Controls layout, colors, fonts, spacing, and overall design.
Images folder – Contains banner, logo, and all images used throughout the site.

This site was built using HTML and external CSS. It includes a navigation menu on all pages, images on each page, consistent design, a table, and a form as required.
